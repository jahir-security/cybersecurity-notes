## Search Skills and Networking - [06-27-2026]

### What This Covers
This module covers essential cybersecurity research skills utilizing open-source intelligence tools and vulnerability databases, seamlessly transitioning into a deep dive of core networking principles. It explores how data travels across the internet, breaking down the OSI and TCP/IP models, network addressing, routing protocols, and the critical differences between secure and insecure protocols used in modern communications.

### Key Concepts
* Utilizing Shodan (tryscan me), VirusTotal (trydetectme), CVE vulnerability databases, technical documentation (man pages), and GitHub for cybersecurity research and threat intelligence.
* Introduction to the Internet and its underlying architecture.
* The OSI model and the specific functions occurring at each of its seven layers.
* The TCP/IP Model and how it compares to OSI architecture.
* IP addressing structures, private vs. public ranges, and Subnetting.
* Differences between connection-oriented (TCP) and connectionless (UDP) protocols.
* Data Encapsulation as information travels through the network layers.
* Telnet and the inherent risks of unencrypted communication.
* Dynamic Host Configuration Protocol (DHCP) for automated IP assignment.
* Address Resolution Protocol (ARP): Bridging Layer 3 IP addressing to Layer 2 MAC addressing.
* Internet Control Message Protocol (ICMP) and utilizing traceroute for network diagnostics.
* Routing mechanisms, pathfinding algorithms, and Network Address Translation (NAT).
* Domain Name System (DNS) hierarchy and how name resolution works.
* Using Whois lookups for domain registration intelligence.
* Insecure application layer protocols, their use cases, default ports, and security problems (HTTP, FTP, SMTP, POP3, IMAP).
* Securing communications using TLS (HTTPS, SMTPS, POP3S, IMAPS).
* Secure remote administration via SSH.
* Secure file transfer alternatives using SFTP and FTPS.
* Virtual Private Networks (VPNs), differentiating between full-tunnel and half-tunnel (split-tunnel) architectures, and the working mechanics of IPSec.

### What I Learned
I developed a strong foundational understanding of how network traffic is routed, encapsulated, and delivered across the internet. I learned to distinguish between the OSI and TCP/IP models, understanding exactly how Layer 2 physical addresses bridge with Layer 3 logical addresses using ARP. Furthermore, I learned the critical security differences between legacy, cleartext protocols and their modern, TLS-encrypted counterparts.
