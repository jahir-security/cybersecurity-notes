# TryHackMe: Core Infrastructure & Networking Mastery
**Timeline:** June 2, 2026 – June 15, 2026  
**Status:** Core Sections Complete

Detailed technical breakdown of the foundational systems architecture, networking protocols, and web mechanics completed during the first phase of the Pre-Security track.

---

## 1. Network Fundamentals
Validated understanding of how data moves across private and public infrastructure.
*   **Network Architecture:** Explored LAN topologies, routing logic, and the core technologies powering private networks.
*   **The OSI Model:** Deep dive into the 7-layer framework to trace data processing from the Physical layer up to the Application layer.
*   **Data Encapsulation:** Analyzed how data is segmented into packets and frames, tracking header attachments (MAC and IP addresses) across the wire.
*   **Network Extension:** Studied firewalls, residential gateways, and the core motivations for expanding local networks safely out to the broader internet.

## 2. How The Web Works
Analyzed the application layer protocols and architectures that govern global web applications.
*   **DNS (Domain Name System):** Mapped out the exact recursive resolution process—from browser cache to Root, TLD, and Authoritative nameservers.
*   **HTTP/HTTPS Protocols:** Broken down request methods (`GET`, `POST`), headers, cookies, and status codes (`200 OK`, `404 Not Found`, `500 Internal Server Error`).
*   **Web Architecture:** Studied how HTML, CSS, and JavaScript interface with back-end databases, detailing the attack surface available to malicious actors during initial reconnaissance.

## 3. Computer Fundamentals
Mastered the baseline hardware and virtualization layers that sustain enterprise IT infrastructure.
*   **System Hardware:** Analyzed core computing components (CPU, RAM, Motherboards, Storage) and how the OS interacts with system hardware.
*   **Client-Server Model:** Examined architecture setups where centralized servers manage and serve resources to distributed clients.
*   **Virtualization:** Studied how hypervisors abstract physical hardware to isolate guest operating systems, paving the way for scalable cloud computing.
*   **Cloud Computing Fundamentals:** Analyzed cloud service models (IaaS, PaaS, SaaS) and deployment structures (Public, Private, Hybrid), focusing on efficiency, elasticity, and shared responsibility security models.
